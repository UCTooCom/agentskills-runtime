
# 编译前的准备工作
1. 把stdx的动态链接库路径指定为环境变量`CANGJIE_STDX_DYNAMIC_PATH`
2. 执行以下命令
```bash
export CANGJIE_STDX_PATH=$CANGJIE_STDX_DYNAMIC_PATH
export POSTGRES=url_of_postgres
```
